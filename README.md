# EcoStep

**Live Site:** [https://eco-step-b2b55.web.app](https://eco-step-b2b55.web.app)

## Features

- Activities to keep users active while being eco friendly
- Various fun and engaging challenges to enroll in and keep the users active
- Monitoring Impact of the challenges in real time
- User has freedom to create challenges as per their needs
- User can also join challenges created by others
- Fully Responsive Design to keep users interactive and ecologically sound across the globe in any platform
